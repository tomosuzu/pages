# レッドビーシュリンプの飼育・繁殖

# 前提

  • 開始時期　2021年4月1日
  • レッドビーシュリンプ5匹　（2021年4月14日に1匹死亡）
  • コトブキ30cmキューブ水槽 (opens new window)
  • ロカボーイ M (opens new window)
  • プラチナソイル (opens new window)
  • 水温の調整は、エアコンにより実施
  • 照明は室内灯、日光。

# 最適な水温

  • 26度以上だと活動が鈍る
  • 24度程度であれば、活動し始める

原因として考えていたのは溶存酸素量の低下だが、

26度以上にした状態でエアレーションを二つにしたとき、

一つの時よりもある程度、活発化したが、明確な活発化には程遠かった。

（活発化：物陰に隠れている個体、時間が少ない）

24度程度にして、エアレーション が二つの時には、

かなり活発化し始めていた。
